About

This is a short Abstraction. View more here.

  • Music Auto generated
  • Hardware Arduino & Pi
  • Sockets Through IP
  • Heartbeat Sensor for BPM
  • Emotion Recognition

YouMusic incorporates a user's heart rate and mood to compose music tailored to them. Using a heart rate sensor or button, Raspberry Pi, and Python, it converts pulse to beats per minute (BPM) using Fourier transforms. It utilizes a GUI where users can upload a photo of themselves. DeepFace analyzes the emotional state of the face to modify major and minor scales, modulation, and dissonance. These values are sent to Pure Data (Pd) via IP sockets. Pd generates chords as well as sampled drum set notes and updates in real-time to illustrate how the user always feels.
